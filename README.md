# Stock Portfolio Analyzer

A Python-based stock portfolio analyzer using NumPy.

## What it does
- Analyzes 3 stocks with price statistics
- Calculates volatility and daily returns
- Shows correlation between stocks
- Generates buy/sell signals
- Provides portfolio summary

## Requirements
- Python 3.x
- NumPy

## How to run
1. Install NumPy: pip install numpy
2. Clone this repository
3. Run: python stockportfolioanalyzer.py
